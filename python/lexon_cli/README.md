# Lexon CLI (Python wrapper)

This package bundles the `lexc` and `lexc-cli` binaries and exposes them as console scripts.

Install (from wheel artifact):
- pip install lexon-cli-rc1-<version>-py3-none-any.whl

Usage:
- lexc --help
- lexc-cli --help

Note: Binaries are copied into the wheel at build time by the CI workflow.
